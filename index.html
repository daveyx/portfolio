about me

web developer

My name is David and i was born in germany. I love the internet, the web and everything what deals around these topics. As a fullstack engineer i want to be able to understand everything what is necessary to run a web application.
Webservers, coding backend logic and frontend stuff like HTML, CSS, JavaScript is what makes fun for me. Not just since i got in contact with ReactJS, i am interested in UI/UX.

remote worker

I feel at home at so many places in the world. Thats why i want to work independend from location. I have more than 8 years experience working remotely and i love it to live as a digital nomad. With the great tools for colaborative working, nowadays it is not necessary any more for developers to work onsite, although the world seems not yet ready for it.
I like the freedom which is brought by living as a digital nomad.

linux enthusiast

I love Linux and use Ubuntu Linux more than 10 years. I feel like it gives me more freedom than other operating systems, although i must say that it is not perfect. I like it to be able do all stuff on my laptop which i will do on a production system, as most production systems are based on Linux.
Furthermore i love the open source concept. In the code is the truth and if i have access to the code i can find out everything.

digital nomad

As i am mostly independent from place and time, i can work from everywhere.
I can accept working times monday until sunday.
Available for work 01:00am - 05:00pm UTC every day.
